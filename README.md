# Background Creator (Web)

A mobile-friendly web app to generate minimalist gradient wallpapers and export them at common iOS and Android resolutions.

## Features

- Linear, radial, and conic gradient modes
- Three-color editing and curated presets
- Randomize and quick color swap actions
- Export presets for iOS and Android wallpaper sizes
- Custom size export for any resolution

## Run locally

Because this project is plain HTML/CSS/JS, run a local web server from the repo root:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.
